## DATA DICTIONARY - Getting and Cleaning Data Course Project


- Subject

  - ID of one of the 30 participants in the study
  
- Activity
  
  - One of the six activities, performed by each subject. The data was transformed as follows:
      - "1" -> "Walking"
      - "2" -> "Walking Upstairs"
      - "3" -> "Walking Downstairs"
      - "4" -> "Sitting"
      - "5" -> "Standing"
      - "6" -> "Laying"



The following variables underwent the following transformation:

- Starting letter t -> "Time-"
- Starting letter f -> "FFT-"
- Name including "Acc" -> "Acceleration-"
- Name including "Jerk" -> "JerkSignal-"
- Name including "Gyro" -> "Gyroscope-"
- Name including "Mag" -> "Magnitude-"
- Name including "Body" -> "Body-"
- Name including "Gravity" -> "Gravity-"
- Name including "mean()" -> "MEAN"
- Name including "std()" -> "SD"
